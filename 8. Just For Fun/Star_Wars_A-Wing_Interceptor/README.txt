# Instructions

Building instructions contained in awing_instructions.pdf  

The .svg files contain the patterns for this model. They are designed for 1/8" material. The reference square in awing_4.svg should measure 1" x 1"  

The laser guns are 1/4" dowels. The 1/8" barrels drilled down the center are optional but recommended.
