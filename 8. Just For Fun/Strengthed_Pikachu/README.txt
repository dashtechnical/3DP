# Summary

My grand-children love the low-poly Pokemon from FLOWALISTIK, such as the Pikachu at http://www.thingiverse.com/thing:376601.

However, the tail on this model is a little fragile for young children, even when printed in flexible filament, so I added an extra brace at the base.

A very simple OpenSCAD file to add the brace to the original STL file is included.
