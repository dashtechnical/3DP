# Summary

<h3>Summary</h3>
Dyson like bladeless fan, Version 2!
Uses a custom designed impeller to move the air instead of a normal fan. 
Airflow is much higher on this Version 2.

<h3>Things you'll need:</h3>
- Filament
- RS-380 motor (Don't use this motor higher than 10K RPM!)
http://www.ebay.com/itm/MABUCHI-RS-380PH-4045-DC3V-12V-6V-7-2V-16200RPM-High-Speed-Carbon-Brush-DC-Motor-/262570016882
- PWM speed controller 
http://www.ebay.co.uk/itm/12V-36V-Pulse-Width-PWM-DC-Motor-Speed-Controller-Regulator-Switch-12V-24V-3A-/191784615003
- 2x 624 Bearing
http://www.ebay.co.uk/itm/10-PCS-Miniature-Rubber-Sealed-Metal-Shielded-Metric-Radial-Ball-Bearing-Model-/400458020258
- 4x ISO4762 or ISO4014 M4 x 10 bolt
- 4x M4 nut

<h3>Optional Things:</h3>
- A metal shaft, see in pictures. (A normal 3D-printed shaft will work fine but a metal, turned shaft reduces vibration and noise.)
- Dust filter or fine mesh so the impeller doesn't clog up over time.
- Rubber feet
