# Summary

Browse all LEGO® parts, bricks and sets, select what you want and download a zip with all the 3D models. Head to [printabrick.org](https://printabrick.org/), search you favorite LEGO® set and print it on your 3D printer! All models available under libre Creative Commons Attribution license.

# How I Designed This

Created by David Hübner, [PrintABrick](https://printabrick.org/) is a web application that combines data from [LDraw](http://www.ldraw.org/), [Rebrickable](https://rebrickable.com/) and [Brickset](https://brickset.com/). Code is available on [GitHub](https://github.com/hubnedav/PrintABrick).
