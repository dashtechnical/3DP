# Summary

This full cutter uses standard utility blades and was designed to be used on dollar tree foam board. It works great for cutting all the way through DTFB to make a perfect cut every time!!

This foam board cutter is designed to use 1 M3x10mm screw.

I suggest printing this with 10% infill and 10% support.
