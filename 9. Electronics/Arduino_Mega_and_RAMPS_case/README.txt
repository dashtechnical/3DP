# Summary

This is heavily based on http://www.thingiverse.com/thing:863076 but is a complete rework and redraw to accommodate an 80mm fan, more ventilation slots and 5mm more height.

Other changes include the square 'hoops' along each side to cable tie cables to, the 12mm high standoffs, 3mm mounting bolts holes (recessed to prevent short circuits) increased size of the M3 nut holders and larger cable exits.

10/12/2016 - I uploaded an alternative back end for the box as this one gives better ribbon cable access for the LCD display and cable access generally.

Other parts designed for my Prusa i3 steel upgrade are:
