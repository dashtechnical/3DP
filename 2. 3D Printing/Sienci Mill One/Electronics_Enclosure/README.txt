# Summary

This part holds a CNC V3.0 shield and Arduino Uno used to control the Sienci Mill One. It also comes with a spot where you can put on a 40mm fan for additional cooling.

Please note it is designed for Arduino Unos with the USB B type port, but some micro USB type Arduinos may work too.

For more information, please visit www.sienci.com
