# Summary

This part is designed to hold a T8 leadscrew nut for the Sienci Mill One using M3 screws and nuts.

This version comes with nut traps which make it easier to assemble.

Please note that two nut holders and one mirrored nut holder is needed per machine.

Learn more at www.sienci.com
