# Summary

This is a simple wedge I made in Solidworks 2014 to help to sure up my Y motor.  This is for the Rework, not the Vanilla, I3 and was made for the 10MM all threaded rods.  Fits like a glove and works extremely well.  

I was inspired by http://www.thingiverse.com/thing:107062

# Instructions

I used 28% honeycomb infill at .28mm layer height for a 1.75mm J-Head with a .35mm nozzle using Slic3r.
